# research_bot

Инструменты для исследования данных — без торговли, только сбор и анализ.

## chainlink_monitor

Мониторинг Chainlink price feeds на Polygon (BTC, ETH, SOL, XRP).

**Что пишет в лог:**
- Каждый новый roundId — с ценой, изменением и интервалом с предыдущего раунда
- Каждые 5 минут — сводная статистика по всем символам

**Что сохраняет:**
- CSV в `research_bot/data/chainlink_rounds_YYYYMMDD_HHMMSS.csv`

**Формат CSV:**
```
ts_utc, symbol, round_id, price, price_change, price_change_pct, seconds_since_last, updated_at_utc
```

### Запуск

```bash
python3 -m research_bot
```

### Цель исследования

- Понять реальную частоту обновлений Chainlink (~27s heartbeat или быстрее при движении ≥0.5%)
- Измерить типичный размер ценового скачка за один раунд
- Найти паттерны: в какое время суток обновления чаще, как ведут себя разные символы
- Данные для оценки потенциального edge стратегии oracle_arb_bot
